Stompee is a log viewer for Java EE
Stompee use java.util.logging to add a custom Handler that stream log records to a web socket connection.
This allows you to view the log online. You can also change the log level of your logger on the fly.
All log levels are restored when the log viewer are stopped.

Stompee is written by Phillip Kruger. You can follow me on twitter for updates. Any comments / suggestions / feedback / feature requests welcomed on the Github page.

Stomp is the Afrikaans word for log (wooden log), Stompie means cigarette butt.
